Contextual Backgrounds

Use as classes de plano de fundo contextuais para fornecer "significado através das cores".

Observe que você também pode adicionar uma classe .text-* se desejar uma cor de texto diferente:

This text is important.

Este texto é importante

Este texto indica sucesso.

Este texto representa algumas informações.

Este texto representa danger.
