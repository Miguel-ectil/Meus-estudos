#  Tipos de dados

Conforme o ECMAcript standard temos 9 tipos de dados:

* Data types
    * Primitive / primitive value
    * Structural
    * Structural Primitive

## Primitivos

* String
* Number
* Boolean
* undefined
* Symbol
* BigInt

## Estructurais

* Object
    * Array
    * Map
    * Set
    * Date
    * ...
* Function

## Primitivo Estructural / Structural Root Primitive

* null
